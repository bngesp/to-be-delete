# User Flow Diagram - Parcours Utilisateur

## Description

Diagramme des parcours utilisateur principaux pour illustrer les interactions avec la plateforme Waajal Ëlëk.

---

## Prompt IA pour User Flow

```
Create a comprehensive user flow diagram for "Waajal Ëlëk" showing the main user journeys through the military pension platform.

DIAGRAM TYPE: User experience flow diagram with multiple user personas

TITLE: "WAAJAL ËLËK - Parcours Utilisateur Principaux"
Subtitle: "Système de Gestion de Retraite Militaire"

COLOR CODING BY USER TYPE:
- Military Personnel (Adhérent): Green (#0B5345)
- Administrator (Gestionnaire): Blue (#2C3E50) 
- System/Automated: Orange (#E67E22)

MAIN USER FLOWS TO SHOW:

1. NEW MEMBER ONBOARDING FLOW:
   Starting point: "Militaire éligible"
   Steps:
   → Découverte du régime
   → Création de compte (authentification)
   → Vérification identité militaire  
   → Choix options cotisation
   → Simulation pension personnalisée
   → Signature contrat numérique
   → Premier versement
   → Confirmation adhésion
   End point: "Adhérent actif"

2. DAILY MEMBER USAGE FLOW:
   Starting point: "Adhérent connecté"
   Branches:
   → Consultation solde de points
   → Modification cotisations
   → Simulation retraite
   → Téléchargement documents
   → Contact support
   → Déconnexion sécurisée

3. RETIREMENT PROCESS FLOW:
   Starting point: "Approche de la retraite"
   Steps:
   → Notification éligibilité
   → Demande de liquidation
   → Choix modalités (rente/capital)
   → Validation documents
   → Calculs actuariels automatiques
   → Approbation gestionnaire
   → Mise en paiement
   End point: "Retraité bénéficiaire"

4. ADMINISTRATOR WORKFLOW:
   Starting point: "Gestionnaire connecté"
   Branches:
   → Validation nouvelles adhésions
   → Monitoring système IA
   → Gestion portefeuille
   → Reporting OHADA
   → Support client niveau 2
   → Configuration paramètres

5. AUTOMATED SYSTEM PROCESSES:
   Starting point: "Déclencheurs automatiques"
   Processes:
   → Calculs cotisations mensuelles
   → Conversion en points automatique
   → Détection fraude IA
   → Recommendations robo-advisor
   → Simulations Monte Carlo
   → Backup et sécurité
   → Notifications push

VISUAL ELEMENTS:
- User persona icons for each flow type
- Decision diamonds for choice points
- Process rectangles for actions
- Database symbols for data operations
- Mobile/web icons for channel indicators
- Security checkpoints (shield icons)
- Automated process indicators (robot icons)
- Success/error states clearly marked

INTERACTION TOUCHPOINTS:
- Web portal interactions
- Mobile app interactions  
- Email notifications
- SMS confirmations
- PDF document generation
- Push notifications
- Phone support escalation

TECHNICAL INTEGRATIONS SHOWN:
- Authentication via Keycloak
- Database updates (PostgreSQL)
- Message queue processing (RabbitMQ)
- AI/ML model interactions
- External API calls (Mobile Money)
- Document generation services
- Audit logging touchpoints

DESIGN SPECIFICATIONS:
- Clean, professional flowchart style
- Consistent iconography throughout
- French language labels
- Clear directional arrows
- Decision points clearly marked
- Error/exception paths included
- Time estimates for each major step
- Channel indicators (web/mobile/email)
- Security validation points highlighted

LAYOUT:
- Landscape orientation for complex flows
- Logical left-to-right progression
- Grouped by user type with visual separation
- Cross-references between flows where users interact
- Legend explaining symbols and color coding

ANNOTATIONS:
- Key performance indicators (time to complete)
- Security checkpoints marked
- Integration points with external systems
- Escalation paths for issues
- Offline capability indicators where relevant

OUTPUT: Professional user journey flowchart suitable for UX documentation and stakeholder presentations
```
