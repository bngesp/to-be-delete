# OFFRE TECHNIQUE COMBINÉE - WAAJAL ËLËK
## Plateforme de Gestion du Régime de Retraite Complémentaire des Armées

---

## I. RÉSUMÉ EXÉCUTIF

Cette offre combine l'expertise actuarielle de FSY CORP avec l'approche modulaire complète de Diamono-Tech pour livrer une solution robuste, sécurisée et évolutive répondant à 100% aux exigences du TDR.

**Objectif :** Développer la plateforme Waajal Ëlëk pour gérer le régime de retraite complémentaire à points par capitalisation destiné aux militaires.

**Date de livraison garantie :** 1er novembre 2025

---

## II. ARCHITECTURE TECHNIQUE HYBRIDE

### 2.1 Stack Technologique Optimisée
- **Backend :** Ruby on Rails 8 (FSY CORP) + API REST sécurisées
- **Base de données :** PostgreSQL avec réplication et sauvegardes automatiques
- **Frontend Web :** React + TypeScript + Tailwind CSS (responsive)
- **Mobile :** PWA + Application native React Native/Flutter
- **Infrastructure :** Docker + Kubernetes + Cloud souverain
- **Paiements :** Intégration Wave, Orange Money, Free Money, virements bancaires

### 2.2 Sécurité Renforcée
- Authentification multi-facteurs (2FA/OTP)
- Chiffrement SSL/TLS bout en bout
- Conformité RGPD et réglementations sénégalaises
- Audit trail exhaustif avec PaperTrail
- Protection OWASP Top 10

## III. MODULES FONCTIONNELS (20 MODULES INTÉGRÉS)

### 3.1 Modules de Base (Phase 1 - Novembre 2025)

#### Module 1 : Gestion du Personnel Militaire
- Dossiers complets : identité, carrière, situation familiale
- Profil professionnel : grade, échelon, unité, limite d'âge
- Rémunération de référence pour calculs
- Import/Export CSV/Excel + interopérabilité paie

#### Module 2 : Gestion des Adhésions
- Cycle complet : création, activation, suspension, résiliation
- Contrats paramétrables : taux, plafonds, options spécifiques
- Workflow de validation multi-niveaux
- Attribution automatique de statuts

#### Module 3 : Gestion des Cotisations (Régime par Points)
- Calcul automatique : base salaire + primes, application taux
- Conversion en points selon valeur d'achat paramétrable
- Échéancier automatique avec gestion retards/pénalités
- Points gratuits et bonifications

#### Module 4 : Tableau de Bord Stratégique
- Indicateurs temps réel : actifs, retraités, cotisations, pensions
- Graphiques évolution : cotisations/pensions mensuelles
- Alertes automatiques : retards, échéances, anomalies
- Drill-down vers détails opérationnels

#### Module 5 : Portail Personnel Militaire
- Accès sécurisé avec authentification forte
- Suivi cotisations, points acquis, simulations
- Documents personnels et notifications
- Interface responsive mobile-first

#### Module 6 : Administration & Sécurité
- Gestion utilisateurs et rôles granulaires
- Paramètres système : valeur point, taux, règles
- Journal de sécurité exhaustif
- Tableau de bord sécurité et audit

### 3.2 Modules Avancés (Phase 2 - Décembre 2025)

#### Module 7 : Gestion des Pensions de Retraite
- Liquidation droits : points × valeur de service
- Options sortie : rente viagère, capital unique, mixte
- Gestion bénéficiaires et réversion
- Ordres de paiement automatisés

#### Module 8 : Personnel Retraité
- Transfert statut Actif → Retraité automatique
- Suivi pensions et historique versements
- Gestion ayants droit et coordonnées bancaires
- Alertes certificats de vie

#### Module 9 : Démissions & Rachats
- Gestion démissions avec impact sur droits
- Rachats partiels/totaux avec conditions d'éligibilité
- Calcul automatique : capital + intérêts - pénalités
- Workflow validation et traçabilité

#### Module 10 : Comptabilité & Comptes
- Enregistrement flux financiers complets
- Gestion multi-comptes bancaires
- Import/Export pour rapprochement
- Journal d'audit financier

#### Module 11 : Rapports & Analyses
- Rapports standard : cotisations, pensions, adhésions
- Analyses ratios : actifs/retraités, taux couverture
- Exports PDF/Excel institutionnels
- Rapports ad hoc personnalisables

### 3.3 Modules Experts (Phase 3 - Janvier 2026)

#### Module 12 : Simulateur Global Analytique (Moteur Actuariel)
- Tables de mortalité intégrées
- Projections long terme (10-30 ans)
- Stress tests et analyses sensibilité
- Monte Carlo pour estimation risques

#### Module 13 : Avances sur Capital
- Demandes en ligne avec vérification éligibilité
- Échéancier remboursement automatique
- Simulation impact capital futur
- Suivi soldes et alertes défauts

#### Module 14 : Gestion des Investissements
- Portefeuille multi-classes : immobilier, actions, obligations
- Suivi performances et calcul gains/pertes
- Analyse risques et alertes seuils
- Reporting consolidé

#### Module 15 : Intérêts & Rendements
- Calcul automatique périodique
- Répartition proportionnelle par adhérent
- Historique multi-années
- Intégration données investissements

#### Module 16 : Gestion des Assurances
- Contrats collectifs/individuels
- Suivi primes et sinistres
- Gestion bénéficiaires
- Intégration module Pensions

#### Module 17 : Centre d'Aide
- Base documentaire organisée par thèmes
- Moteur recherche plein texte
- FAQ évolutive
- Accès différencié par profil

#### Module 18 : Simulations Personnalisées
- Préremplissage données militaire
- Scénarios multiples : normal, accéléré, anticipé
- Visualisations graphiques évolution
- Recommandations optimisation

#### Module 19 : Application Mobile & API
- Application native iOS/Android
- Mode offline avec synchronisation
- Paiements intégrés mobile money
- Notifications push sécurisées

#### Module 20 : Moteur de Calcul Actuariel Avancé
- Intégration complète tables mortalité
- Hypothèses multiples (optimiste/réaliste/pessimiste)
- Calculs rentes viagères précis
- API pour autres modules
